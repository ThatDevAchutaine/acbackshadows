## Introduction

> Allows you to get great shadows every time!

## Code Samples

```
import { acbackshadows } from 'acbackshadows';

acbackshadows({
    shadow__type: 'hard',
    padding: true
});
```

## Installation

> Simple to install just open cmd and type "npm i acbackshadows --save"
